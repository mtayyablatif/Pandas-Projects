## What this repo covers
| # | Project | Core Skills |
|---|---|---|
| 1 | First Look | `.shape`, `.dtypes`, `.describe()`, `.head()`/`.tail()`, `.unique()` |
| 2 | Missing Data | `.isna().sum()`, fillna (median/mode), dropping columns |
| 3 | Filtering | boolean filtering, `.isin()`, `.loc[]`, negation |
| 4 | GroupBy & Aggregation | `.groupby()`, `.agg()`, `.size()` vs `.count()`, `.reset_index()` |
| 5 | Columns, Apply, Pivot | new columns, `.apply()`, `.sort_values()`, `pivot_table()`, exporting to csv/xlsx |

## Dataset used
- **Titanic** — loaded via `seaborn.load_dataset('titanic')`, no download needed
